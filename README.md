# ssc-dashboard-prototype
This app allows Caravan staff to explore visualizations of the data collected from the Safe Shelters Collaborative app.

## Technologies
stack (MEAN): 
  * AngularJ
  * Express
  * MongoDB
  * NodeJS

## Dependencies
* Bootstrap 3.2.0
* JQuery
* moment.js
* daterangepicker.js
* angular-daterangepicker.js
* csvtojson 1.0.0
* mongoose 4.5.1
